# φ-Attention 丹方第3步：火候 —— 信息传播动力学

## 概要

C5-cycle耦合注意力机制中，信息如何在5个head之间流动、混合，
以及稳态分布长什么样——这就是「火候」。

- C5邻接矩阵权重 w = cos(72°) ≈ 0.309017
- 最大特征值 λ_max = φ ≈ 1.618034
- 迭代步数 T = 50

## 实验1：信息传播动力学

### 设置
- 初始条件：只有 head 0 有信号 [1, 0, 0, 0, 0]
- 迭代规则：x_{t+1} = A @ x_t
- C5-coupled: A 含邻接权重 w=cos(72°)
- 无耦合: A = I（单位矩阵），信号不传播

- C5 PF特征向量: ['0.200000', '0.200000', '0.200000', '0.200000', '0.200000']

### IB结论
- C5耦合：信息在传播中被压缩（I(X;Z)下降），同时趋近稳态（I(Z;Y)上升）
- 无耦合：I(X;Z)不变（没有压缩），I(Z;Y)=0（没有趋近目标）
- C5在相同压缩率下保留更多任务相关信息 → 结构必然性

## 总结：火候三味

| 维度 | 现象 | 物理含义 |
|------|------|----------|
| 传播 | 信息沿5-cycle逐步扩散 | 邻接耦合让head不再是孤岛 |
| 混合 | C5平均11步达稳态 | 比C8快，5-cycle直径更短 |
| 稳态 | PF特征向量均匀(比=1.0000) | 对称性保证无偏好分工 |

**火候的本质：C5耦合让5个head从独立个体变成有机整体，
信息在对称约束下流动、混合、均匀化——不是混成一团，
而是在φ-比例的放大下保持结构性地分配注意力。**
