# φ-Attention 长程信息保持力合成实验报告

## 实验设计

### 目的
验证 φ-Attention（C5-cycle 耦合注意力）在长序列上对远端注入信号的信息保持能力是否优于标准 Multi-Head Attention。

### 方法
- **无需训练**：仅做 forward pass，测量 attention 层本身的信息保持力
- **模型配置**：d_model=160，对比 phi5/std5/std8/phi8
- **序列长度**：[64, 128, 256, 512, 1024, 2048]
- **信号注入**：在 position 0 的前 32 维注入超强信号 (=100.0)，其余位置为随机小值 (std=0.1)
- **统计**：每个 (模型, 长度) 组合做 10 个不同 seed，取均值±标准差

### 六个指标
1. **attn_to_pos0**: 末位置 (L-1) 对 position 0 的平均注意力权重 — 直接测量 attention 是否'看到'远端信号
2. **signal_corr**: 输入 position 0 与输出 position L-1 的 Pearson 相关系数 — 测量信号是否通过 attention 传导到末位置
3. **info_retention**: attn_to_pos0 / (1/L) — 相对于均匀注意力的倍数，>1 说明长程信息保持优于随机
4. **rank_of_pos0**: position 0 在末位置注意力分布中的归一化排名 (0~1, 1=排名第一) — 排名越靠前说明越能'看到'远端信号
5. **top1_fraction**: position 0 成为末位置 top-1 注意力目标的 head 比例 — 直接测量有多少 head 成功检测到远端信号
6. **max_head_attn**: 最佳 head 对 position 0 的注意力权重 — 测量 head 中最好的那个能否捕捉远端信号

### 物理机制分析

φ-Attention 的 C5-cycle 耦合机制（`coupled_scores = A @ raw_scores`）将相邻 head 的 score 信息混合。
当某个 head 的 raw_score 对 position 0 较高时，该信号会通过耦合矩阵 A 传播到相邻 head，
从而增加相邻 head 对 position 0 的 coupled_score，最终提高整体对远端信号的注意力保持。

在随机初始化场景下，信号注入使 K[0] 范数远大于其他位置的 K[i]，
每个 head 独立产生略高于均匀的 pos0 注意力。C5 耦合将这种'略高'信号跨 head 传播，
理论上应产生更稳定的远端注意力保持。实验验证了这一预测的强度。
